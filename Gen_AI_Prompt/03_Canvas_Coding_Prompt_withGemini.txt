너는 바닐라 HTML/CSS/JavaScript 로 반응형 원페이지 웹사이트를 만드는 프론트엔드 개발자야. 
프레임워크나 외부 빌드 도구 없이 순수 코드로만 작업해줘.

아래 대상을 소개하는 1 페이지 웹사이트 (SPA)를 만들어줘.
(코드가 무엇을 설명하는지 주석으로 설명해줘.)

-대상: 황규진 (나)
{나이(생년월일): 28(대한민국 나이 측정 방식) (1999.09.02)},
{거주지: 경기도 부천시},
{취미: 영화 시청(특히, 공포 영화), 음악 감상, 운동(헬스)},
{과거 취미: 컴퓨터 게임, 볼링}
{MBTI: INFP}
{이력서: 소프트웨어학과 전공, 게임 개발자(클라이언트) 준비, Unity3D, Unreal Engine, C#, C++}

- 추가적으로 소개하고 싶은 내용:
{백엔드개발자로 전향을 생각했고, SSAFY 교육을 통해 사람 구실 할 수 있는 백엔드개발자로 성장하고 싶음}
{공백기동안 무엇을 했나: 20kg 감량}
{재밌게 본 영화, 요즘 많이 듣는 음악, 재밌게 본 애니메이션 하나씩 소개하고 싶음}
{영화: 군체, 음악: 0+0, 애니메이션: 진격의 거인} 
{모두에게 하고싶은 말: 정말 많이 부족하지만 그만큼 배우고자 노력하겠습니다.}
{모두에게 하고싶은 말2: 집에서 사과게임 같이 할 사람 MM 주세요} 

이 내용을 보고 브랜딩 기획자가 컨셉을 정리해줬어. 웹사이트 제작에 참고해줘.
-캐치프레이즈,"20kg의 집념으로, 이제는 탄탄한 백엔드를 빌드합니다."

-핵심 매력 포인트
• [실행력] 20kg 감량으로 증명한 끈기
• [성장 잠재력] SSAFY를 통해 도약할 준비가 된 몰입도
• [친근한 소통력] 사과게임과 취미를 공유하는 유쾌한 동료미

-타겟 방문자
• 주 타겟: SSAFY 동료 및 교수진/멘토 (협업하고 싶은 동료로 어필)
부 타겟: 미래의 IT 기업 인사담당자 및 팀장 (끈기 있는 인재로 어필)

-스토리텔링 흐름
1. Intro: 캐치프레이즈를 활용한 강렬한 첫인상
2. Identity: 20kg 감량 스토리로 끈기 증명 + 백엔드 전향 계기
3. Taste: 규진's Pick (영화, 음악, 애니메이션 추천)
4. Outro: 겸손한 포부(정말 많이 부족하지만 그만큼 배우고자 노력하겠습니다.) + "집에서 사과게임 같이 할 사람 MM 주세요" 마무리

-무드 / 톤,
#대담한 (Bold) (굵은 글씨, 확실한 대비)
#솔직한 (Candid) (담백한 레이아웃, 여백 활용)
#위트있는 (Witty) (센스 있는 문구와 친근한 요소)"

-비주얼 요소
• 컬러: 딥 차콜 & 제트 블랙 (배경/기본) + 애플 레드 (포인트 🍎)
• 폰트: 제목용 굵은 고딕(Pretendard Bold 등) + 본문용 깔끔한 고딕


페이지는 아래 순서로 구성해줘. (사진이 필요한 부분은 우선 회색 placeholder로 해줘. 내가 직접 수정할 수 있도록 코딩해줘.)
  1. 히어로 - "황규진" 타이틀 + 캐치프레이즈 + 첫인상용 그래픽 영역
  2. 소개 - "황규진은 어떤 사람인가요?"
  3. 매력 포인트 카드 3 개 (실행력 / 성장 잠재력 / 친근한 소통력) (카드는 뒷면으로 뒤집어져 있고 클릭했을 때 뒤집혀서 내용이 보이게 해줘.)
  4. 스토리텔링 흐름 섹션 - "Intro -> Identity -> Taste -> Outro" 로 이어지는 흐름을 타임라인 형태로 보여주기
  5. Taste 부분은 내가 소개하는 것들을 사진과 함께 설명할 수 있도록 해줘.
  6. 갤러리 - 관련 이미지 6 장 그리드 (우선 회색 placeholder 로)
  7. CTA - 카드를 버튼으로. 클릭하면 살짝 확대되는 반응 효과를 주고 내용이 표시된 카드 앞면으로 부드럽게 회전시켜줘. 앞면이 된 카드를 다시 클릭하면 뒷면으로 다시 부드럽게 회전시켜줘.
